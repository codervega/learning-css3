CSS FLEX

THE FLEX or flexible box model in css is a one dimensional layout model that has flexible and effiecient layouts with distributed spaces.

Flexible Box layout Module, make it easier to design flexible responsive layput struture without using float or positioning.

properties for the Parent(flex container)

1) display:flex;

2) flex-direction:row;

3)flex-wrap: wrap;

4)flex-flow:row wrap (flex-flow=flex-direction+flex-wrap;)

5)justify-content:center;

6)align-item:center;

7)column-gap:20px

8)row-gap:20px;

9)gap:20px;

Properties for thw Children (Flex Item)

1)flex:1; or flex-grow:1;

2)FLEX-SHRINK:3;

3)order:2;

4)align-self:center;
